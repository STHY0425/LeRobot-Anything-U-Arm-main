## 🎯 核心原理

### 基本概念

舵机（Servo）内部有一个**位置传感器**（通常是电位器或编码器），可以实时反馈当前的转动角度。这个项目通过**串口通信**读取舵机的位置信息，然后转换为角度值。

### 工作原理图

```
┌─────────────┐      串口命令       ┌──────────────┐
│   电脑/树莓派  │ ──────────────→   │   舵机控制板   │
│             │                    │              │
│ servo_reader│ ←──────────────   │  (主控芯片)   │
│   .py       │      位置反馈       │              │
└─────────────┘                    └──────┬───────┘
                                          │
                                          │ 控制信号
                                          ↓
                                   ┌──────────────┐
                                   │   舵机本体    │
                                   │              │
                                   │ ┌──────────┐ │
                                   │ │位置传感器 │ │
                                   │ │(电位器)  │ │
                                   │ └──────────┘ │
                                   └──────────────┘
```

## 🔌 两种舵机方案

### 方案 1: 中菱舵机（Zhonglin Servo）

#### 通信协议
- **接口**: 串口 (Serial)
- **波特率**: 115200
- **协议**: 自定义文本协议

#### 角度读取流程

```python
# 1. 发送读取命令
self.ser.write('#001PRAD!'.encode('ascii'))
#              ^^^  ^^^^
#              |    |
#              |    +-- 命令: PRAD (Position Read)
#              +------- 舵机ID: 001

# 2. 接收响应
response = self.ser.read_all().decode('ascii')
# 响应格式: "P1500" (PWM 值)

# 3. 解析 PWM 值
match = re.search(r'P(\d{4})', response)
pwm_val = int(match.group(1))  # 例如: 1500
```

#### PWM 到角度的转换

```python
def pwm_to_angle(pwm_val, pwm_min=500, pwm_max=2500, angle_range=270):
    """
    PWM 值转换为角度
    
    原理:
    - 舵机内部使用 PWM (脉宽调制) 控制位置
    - PWM 值范围: 500-2500 (对应舵机的物理行程)
    - 角度范围: 0-270° (舵机的转动范围)
    
    转换公式:
    angle = (pwm_val - pwm_min) / (pwm_max - pwm_min) * angle_range
    """
    pwm_span = pwm_max - pwm_min  # 2000
    angle = (pwm_val - pwm_min) / pwm_span * angle_range
    return angle

# 示例:
# PWM = 500  → angle = 0°
# PWM = 1500 → angle = 135°
# PWM = 2500 → angle = 270°
```

#### 完整代码示例

```python
class ServoReaderNode:
    def __init__(self):
        # 打开串口
        self.ser = serial.Serial('/dev/ttyUSB0', 115200, timeout=0.1)
        
        # 初始化: 读取零位
        self.zero_angles = [0.0] * 7
        for i in range(7):
            response = self.send_command(f'#{i:03d}PRAD!')
            angle = self.pwm_to_angle(response)
            self.zero_angles[i] = angle
    
    def run(self):
        while True:
            for i in range(7):
                # 读取当前位置
                response = self.send_command(f'#{i:03d}PRAD!')
                angle = self.pwm_to_angle(response)
                
                # 计算相对零位的偏移
                angle_offset = angle - self.zero_angles[i]
                
                # 发布到 ROS 话题
                self.pub.publish(Float64MultiArray(data=angle_offset))
```

### 方案 2: 飞特舵机（Feetech Servo）

#### 通信协议
- **接口**: 串口 (Serial)
- **波特率**: 1000000 (1Mbps)
- **协议**: Dynamixel 兼容协议（二进制）

#### 角度读取流程

```python
# 1. 使用 SDK 读取位置寄存器
# SMS_STS_PRESENT_POSITION_L = 56 (寄存器地址)

# 2. 批量读取多个舵机
groupSyncRead = GroupSyncRead(packetHandler, 
                              SMS_STS_PRESENT_POSITION_L, 
                              4)  # 读取 4 字节

# 3. 添加要读取的舵机 ID
for i in range(7):
    groupSyncRead.addParam(i + 1)  # ID: 1-7

# 4. 发送读取命令并接收
groupSyncRead.txRxPacket()

# 5. 获取每个舵机的位置
for i in range(7):
    current_pos = groupSyncRead.getData(i + 1, 
                                       SMS_STS_PRESENT_POSITION_L, 
                                       2)  # 读取 2 字节
```

#### 位置值到角度的转换

```python
def position_to_angle(current_pos, zero_pos):
    """
    位置值转换为角度
    
    原理:
    - 飞特舵机使用 12 位编码器
    - 位置范围: 0-4095 (2^12 - 1)
    - 中心位置: 2047
    - 一圈对应: 4096 个位置值
    
    转换公式:
    angle = (current_pos - zero_pos) / 4096.0 * 360.0
    """
    angle = (current_pos - zero_pos) / 4096.0 * 360.0
    return angle

# 示例:
# pos = 0    → angle = -180° (相对零位)
# pos = 2047 → angle = 0°
# pos = 4095 → angle = +180°
```

#### 零位校准原理

飞特舵机支持**硬件零位校准**：

```python
def calibrate_zero_position(scs_id):
    """
    零位校准流程
    
    1. 解锁 EPROM (可写入非易失性存储器)
    2. 读取当前位置 (raw_pos)
    3. 计算偏移量: offset = raw_pos - 2047
    4. 写入偏移量到 EPROM
    5. 锁定 EPROM
    
    效果: 舵机会记住当前位置为零位，断电后仍保持
    """
    # 1. 解锁
    packetHandler.unLockEprom(scs_id)
    
    # 2. 读取当前位置
    raw_pos = packetHandler.read2ByteTxRx(scs_id, 
                                          SMS_STS_PRESENT_POSITION_L)
    
    # 3. 计算偏移
    offset = raw_pos - 2047
    
    # 4. 编码偏移量（处理负数）
    if offset < 0:
        encoded_offset = (1 << 11) | abs(offset)  # 最高位为符号位
    else:
        encoded_offset = offset
    
    # 5. 写入 EPROM
    packetHandler.write2ByteTxRx(scs_id, SMS_STS_OFS_L, encoded_offset)
    
    # 6. 锁定
    packetHandler.LockEprom(scs_id)
```

## 🔑 关键技术点

### 1. 零位校准

**为什么需要零位校准？**

- 每次启动时，舵机的初始位置可能不同
- 需要记录初始位置作为"零点"
- 后续所有角度都是相对这个零点的偏移

```python
# 启动时记录零位
self.zero_angles[i] = read_current_angle(i)

# 运行时计算偏移
angle_offset = current_angle - self.zero_angles[i]
```

### 2. 平滑插值

**为什么需要插值？**

- 舵机读取可能有噪声
- 避免角度突变导致机械臂抖动
- 提供更平滑的控制体验

```python
# 懒惰插值算法
target_angle = read_servo()
current_angle = published_angle

for step in range(5):
    delta = target_angle - current_angle
    current_angle += delta * 0.2  # 每次移动 20%
    publish(current_angle)
    sleep(0.02)  # 50 Hz
```

### 3. 异常检测

**检测角度跳变：**

```python
if abs(new_angle - last_angle) > 90:
    raise Warning("角度跳变过大，可能硬件故障")
```

### 4. 通信优化

**中菱舵机（文本协议）：**
- 优点：易于调试，人类可读
- 缺点：速度较慢，需要解析文本

**飞特舵机（二进制协议）：**
- 优点：速度快，支持批量读取
- 缺点：协议复杂，需要 SDK

## 🔬 技术细节

### PWM 原理

```
PWM (脉宽调制):
    ┌─┐     ┌─┐     ┌─┐
    │ │     │ │     │ │
────┘ └─────┘ └─────┘ └────
    ↑       ↑       ↑
    |       |       |
  500μs  1500μs  2500μs
    |       |       |
    0°     135°    270°

脉宽越长 → 舵机转动角度越大
```

### 编码器原理

```
12位编码器:
- 分辨率: 2^12 = 4096 个位置
- 一圈: 360° / 4096 ≈ 0.088° 每步
- 范围: 0-4095

位置值 → 角度:
angle = (position / 4096) * 360°
```

## 💡 总结

舵机角度读取的核心原理：

1. **硬件层面**: 舵机内部有位置传感器（电位器/编码器）
2. **通信层面**: 通过串口发送命令，读取位置值
3. **转换层面**: 将 PWM/位置值转换为角度
4. **校准层面**: 记录零位，计算相对偏移
5. **优化层面**: 插值平滑，异常检测

这样就实现了实时、准确、平滑的舵机角度读取！
