# 🚀 中菱舵机快速参考

## ❓ 能直接用吗？

### ❌ 不能！必须先配置 ID

**原因**: 出厂舵机 ID 都是 000，会同时响应命令

---

## ⚡ 快速配置（3步）

### 1️⃣ 运行配置工具

```bash
cd src/uarm/scripts/Uarm_teleop/Zhonglin_servo
python3 setup_servos.py
```

### 2️⃣ 选择选项 1（配置 ID）

```
请选择操作:
  1. 配置舵机 ID (首次使用必须)  ← 选这个
  2. 测试所有舵机
  ...
```

### 3️⃣ 按提示逐个配置

```
1. 只连接第 1 个舵机 → 配置为 ID 0
2. 断开，连接第 2 个舵机 → 配置为 ID 1
3. 重复直到 ID 6
```

---

## ✅ 验证配置

```bash
# 连接所有舵机
python3 setup_servos.py

# 选择选项 2（测试）
# 应该看到 7 个不同的角度值
```

---

## 📋 配置清单

- [ ] 已配置 ID 0-6（7个舵机）
- [ ] 测试通过（7/7 响应）
- [ ] 串口正确（/dev/ttyUSB0）
- [ ] 波特率正确（115200）

---

## 🎯 开始使用

```bash
# 终端 1
roscore

# 终端 2
rosrun uarm servo_reader.py

# 终端 3
python3 servo2realman.py
```

---

## 💡 记住

- ✅ **配置一次，永久有效**（断电不丢失）
- ⚠️ **必须逐个配置**（不能同时连接）
- 🔧 **配置后才能用**（否则无法单独控制）
