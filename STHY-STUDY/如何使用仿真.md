# 🎮 LeRobot Anything 仿真使用指南

## ✅ 环境已配置完成！

所有依赖已成功安装，你现在可以开始使用仿真环境了。

## 🚀 快速开始

### 1. 查看静态机器人模型

```bash
cd src/simulation

# 查看 Franka Panda 机械臂（7轴）
python3 static_robot_viewer.py --robot panda

# 查看 XArm6 + Robotiq 夹爪
python3 static_robot_viewer.py --robot xarm6_robotiq

# 查看 ARX-X5（6轴）
python3 static_robot_viewer.py --robot arx-x5

# 查看 SO-100（5轴）
python3 static_robot_viewer.py --robot so100

# 查看 Piper（6轴）
python3 static_robot_viewer.py --robot piper

# 查看 XLeRobot（双臂）
python3 static_robot_viewer.py --robot x_fetch
```

### 2. 控制参数

```bash
# 指定运行时间（秒）
python3 static_robot_viewer.py --robot panda --duration 10

# 指定初始姿态
python3 static_robot_viewer.py --robot panda --pose home

# 指定场景
python3 static_robot_viewer.py --robot panda --scene Empty-v1
```

### 3. 查看帮助信息

```bash
python3 static_robot_viewer.py --help
```

## 🎯 操作说明

运行仿真后，你会看到一个 3D 渲染窗口：

- **鼠标左键拖动**：旋转视角
- **鼠标右键拖动**：平移视角
- **鼠标滚轮**：缩放
- **Ctrl+C**：退出程序

### 3. 自定义脚本

创建你自己的仿真脚本：

```python
import gymnasium as gym
import mani_skill.envs
import numpy as np

# 创建环境
env = gym.make(
    "Empty-v1",
    robot_uids="panda",
    obs_mode="none",
    render_mode="human"
)

# 重置环境
obs, info = env.reset()

# 控制循环
for i in range(1000):
    # 生成动作（关节角度）
    action = env.action_space.sample()
    
    # 执行动作
    obs, reward, terminated, truncated, info = env.step(action)
    
    # 渲染
    env.render()
    
    if terminated or truncated:
        obs, info = env.reset()

env.close()
```

## 🐛 常见问题

### 1. Vulkan 警告

如果看到 "Failed to find Vulkan ICD file" 警告：

```bash
# 安装 NVIDIA 驱动（如果你有 NVIDIA 显卡）
sudo apt install -y nvidia-driver-535 nvidia-utils-535

# 或者安装 Mesa 驱动
sudo apt install -y mesa-vulkan-drivers

# 重启后生效
sudo reboot
```

这个警告不影响使用，仿真仍然可以正常运行。

### 2. 窗口无法显示

如果是远程服务器或没有图形界面：

```python
# 修改 render_mode 为离屏渲染
env = gym.make(..., render_mode="rgb_array")
```

### 3. 性能问题

```bash
# 降低渲染频率
python3 static_robot_viewer.py --robot panda --rate 10
```
