# 🔧 中菱舵机安装顺序和ID对应关系

## ❓ 安装是从下到上ID从小到大吗？

### ✅ 是的！

**舵机 ID 从下到上（从基座到末端）依次为：0 → 1 → 2 → 3 → 4 → 5 → 6**

## 🎯 详细对应关系

### 标准 6 轴机械臂 + 夹爪配置

| ID | 位置 | 关节名称 | 英文名称 | 运动方式 | 说明 |
|----|------|---------|---------|---------|------|
| **0** | 最底部 | 基座旋转关节 | Base Joint | 水平旋转 | 整个机械臂左右旋转 |
| **1** | 第二层 | 肩关节 | Shoulder Joint | 俯仰运动 | 大臂上下摆动 |
| **2** | 第三层 | 肘关节 | Elbow Joint | 俯仰运动 | 小臂弯曲伸展 |
| **3** | 第四层 | 腕关节1 | Wrist Yaw | 水平旋转 | 手腕左右转动 |
| **4** | 第五层 | 腕关节2 | Wrist Pitch | 俯仰运动 | 手腕上下摆动 |
| **5** | 第六层 | 腕关节3 | Wrist Roll | 滚转运动 | 手腕自转 |
| **6** | 最顶部 | 夹爪 | Gripper | 开合运动 | 抓取物体 |

## 🔍 如何确认安装顺序

### 方法 1: 物理位置判断

```bash
# 从下往上数：
1. 最接近地面/底座的 → ID 0
2. 往上第二个 → ID 1
3. 往上第三个 → ID 2
4. 往上第四个 → ID 3
5. 往上第五个 → ID 4
6. 往上第六个 → ID 5
7. 最顶端（夹爪）→ ID 6
```

### 方法 2: 运动链判断

```
基座 → 肩 → 肘 → 腕1 → 腕2 → 腕3 → 夹爪
ID0   ID1   ID2   ID3    ID4    ID5    ID6
```

查看 `servo_zero.py` 中的初始位置：

## 🛠️ 实际安装步骤

### 步骤 1: 组装机械臂

```
1. 安装基座舵机（ID 0）到底板
   ↓
2. 在 ID 0 上安装肩关节舵机（ID 1）
   ↓
3. 在 ID 1 上安装肘关节舵机（ID 2）
   ↓
4. 在 ID 2 上安装腕关节1舵机（ID 3）
   ↓
5. 在 ID 3 上安装腕关节2舵机（ID 4）
   ↓
6. 在 ID 4 上安装腕关节3舵机（ID 5）
   ↓
7. 在 ID 5 上安装夹爪舵机（ID 6）
```

### 步骤 2: 配置舵机 ID

**重要**: 在组装之前或组装过程中配置 ID！

```bash
# 方法 A: 组装前配置（推荐）
# 1. 逐个配置所有舵机的 ID（0-6）
# 2. 在每个舵机上贴标签标注 ID
# 3. 按照 ID 顺序组装

# 方法 B: 组装时配置
# 1. 安装第一个舵机（底座）
# 2. 配置为 ID 0
# 3. 安装第二个舵机（肩关节）
# 4. 配置为 ID 1
# 5. 依此类推...
```

## ⚠️ 常见错误

### 错误 1: ID 顺序颠倒

```
❌ 错误: 从上到下配置 ID 0-6
✅ 正确: 从下到上配置 ID 0-6

如果配置反了：
- 控制基座时，夹爪会动
- 控制夹爪时，基座会动
- 运动方向完全混乱
```

### 错误 2: 跳过某些 ID

```
❌ 错误: ID 0, 1, 3, 4, 5, 6（跳过 ID 2）
✅ 正确: ID 0, 1, 2, 3, 4, 5, 6（连续）

如果跳过 ID：
- 代码会尝试读取不存在的 ID
- 导致通信超时或错误
```

### 错误 3: 重复 ID

```
❌ 错误: 两个舵机都是 ID 1
✅ 正确: 每个舵机唯一的 ID

如果 ID 重复：
- 两个舵机会同时响应
- 无法单独控制
```

### 单独测试每个舵机

```bash
# 创建测试脚本
nano test_single_servo.py
```

```python
#!/usr/bin/env python3
import serial
import time
import re

SERIAL_PORT = '/dev/ttyUSB0'
BAUDRATE = 115200

def pwm_to_angle(response_str):
    match = re.search(r'P(\d{4})', response_str)
    if not match:
        return None
    pwm_val = int(match.group(1))
    angle = (pwm_val - 500) / 2000 * 270
    return angle

def test_servo(servo_id):
    with serial.Serial(SERIAL_PORT, BAUDRATE, timeout=0.5) as ser:
        print(f"\n测试舵机 ID {servo_id}")
        print("=" * 40)
        
        for _ in range(10):
            cmd = f'#{servo_id:03d}PRAD!\r\n'.encode()
            ser.write(cmd)
            time.sleep(0.1)
            
            response = ser.read_all().decode()
            angle = pwm_to_angle(response)
            
            if angle is not None:
                print(f"角度: {angle:6.1f}°", end='\r')
            else:
                print(f"无响应", end='\r')
            
            time.sleep(0.1)
        
        print()  # 换行

if __name__ == '__main__':
    print("舵机单独测试工具")
    print("=" * 40)
    print("请手动移动舵机，观察角度变化")
    print()
    
    for i in range(7):
        input(f"按 Enter 测试 ID {i}...")
        test_servo(i)
        print(f"✅ ID {i} 测试完成")
```

```bash
# 运行测试
python3 test_single_servo.py

# 按提示操作：
# 1. 按 Enter 开始测试 ID 0
# 2. 手动移动最底部的舵机
# 3. 如果角度值变化 → ID 0 正确 ✅
# 4. 如果角度值不变 → ID 0 配置错误 ❌
# 5. 重复测试 ID 1-6
```
