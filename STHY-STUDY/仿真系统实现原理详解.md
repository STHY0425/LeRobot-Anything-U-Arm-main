## 🔄 仿真流程

### 完整的仿真循环

```
初始化 → 重置环境 → [执行动作 → 物理步进 → 渲染 → 收集观察] → 关闭
         ↑                                                    ↓
         └────────────────── 循环 ──────────────────────────┘
```

### 详细步骤分解

#### 1. 初始化阶段

```python
# 步骤 1.1: 创建物理场景
scene = sapien.Scene()
scene.set_timestep(1/240)  # 设置时间步长

# 步骤 1.2: 加载机器人 URDF
loader = scene.create_urdf_loader()
robot = loader.load("panda.urdf")

# 步骤 1.3: 配置控制器
controller = PDJointPosController(
    joints=robot.get_active_joints(),
    stiffness=1e3,
    damping=1e2
)

# 步骤 1.4: 设置渲染器
renderer = sapien.SapienRenderer()
scene.set_renderer(renderer)
```

#### 2. 重置阶段

```python
def reset(self):
    # 2.1: 重置机器人关节位置
    qpos = np.array([0, 0, 0, -1.57, 0, 1.57, 0.78, 0.04, 0.04])
    self.robot.set_qpos(qpos)
    
    # 2.2: 重置机器人速度
    self.robot.set_qvel(np.zeros(9))
    
    # 2.3: 重置物体位置（如果有）
    self.cube.set_pose(sapien.Pose([0.5, 0, 0.05]))
    
    # 2.4: 物理场景稳定
    for _ in range(10):
        self.scene.step()
    
    # 2.5: 收集初始观察
    obs = self._get_obs()
    return obs
```

#### 3. 执行动作阶段

```python
def step(self, action):
    # 3.1: 控制器处理动作
    # action: 目标关节位置 [j1, j2, j3, j4, j5, j6, j7, gripper]
    target_qpos = action
    
    # 3.2: PD 控制器计算力矩
    current_qpos = self.robot.get_qpos()
    current_qvel = self.robot.get_qvel()
    
    # PD 控制公式: τ = Kp * (target - current) - Kd * velocity
    torque = self.stiffness * (target_qpos - current_qpos) - \
             self.damping * current_qvel
    
    # 3.3: 应用力矩到关节
    self.robot.set_qf(torque)
    
    # 3.4: 物理仿真步进
    self.scene.step()  # 推进 1/240 秒
    
    # 3.5: 收集观察
    obs = self._get_obs()
    
    # 3.6: 计算奖励
    reward = self._compute_reward()
    
    # 3.7: 检查终止
    terminated = self._check_success()
    truncated = self._check_timeout()
    
    return obs, reward, terminated, truncated, info
```

#### 4. 渲染阶段

```python
def render(self):
    # 4.1: 更新相机位置
    self.camera.set_pose(camera_pose)
    
    # 4.2: 渲染场景

```

## 💻 代码实现分析

### 示例 1: `sin_xarm.py` - 简单运动演示

```python
import gymnasium as gym
import numpy as np
import time

# 创建环境
env = gym.make(
    "PushCube-v1",           # 任务：推方块
    robot_uids="xarm6_robotiq",  # 机器人：XArm6
    render_mode="human",     # 渲染：窗口显示
    control_mode="pd_joint_pos",  # 控制：关节位置
)

# 重置环境
obs, _ = env.reset(seed=0)

# 定义两个姿态
pose_a = np.radians([14.1, -8, -24.7, 196.9, 62.3, -8.8, 0.0])
pose_b = np.radians([-30, -8, 0, 196.9, 62.3, -8.8, 0.0])

# 循环运动
while True:
    # A → B
    for t in np.linspace(0, 1, 200):
        action = (1 - t) * pose_a + t * pose_b  # 线性插值
        env.step(action)  # 执行动作
        env.render()      # 渲染
        time.sleep(0.01)  # 延时
    
    # B → A
    for t in np.linspace(0, 1, 200):
        action = (1 - t) * pose_b + t * pose_a
        env.step(action)
        env.render()
        time.sleep(0.01)
```

**工作原理**：
1. 创建包含 XArm6 机器人和方块的环境
2. 定义两个关节角度姿态
3. 使用线性插值在两个姿态间平滑过渡
4. 每步调用 `env.step()` 推进物理仿真
5. 调用 `env.render()` 更新显示

### 示例 2: `teleop_sim.py` - 遥操作仿真

**系统架构**：

```
串口读取线程 → 角度队列 → 仿真控制线程
    ↓                         ↓
  舵机角度                  机器人动作
    ↓                         ↓
  角度映射                  物理仿真
    ↓                         ↓
  发布到队列                 渲染显示
```

**核心代码**：

```python
class ServoTeleoperatorSim:
    def __init__(self, robot_uids, serial_port):
        # 初始化串口
        self.ser = serial.Serial(serial_port, 115200)
        
        # 创建仿真环境
        self.env = gym.make(
            "ReplicaCAD_SceneManipulation-v1",
            robot_uids=robot_uids,
            render_mode="human",
        )
        
        # 线程安全队列
        self.arm_pos_queue = Queue(maxsize=1)
        
        # 创建生产者线程（读取舵机）
        self.produce_thread = Thread(
            target=self.angle_stream_loop,
            daemon=True
        )
        
        # 创建消费者线程（控制仿真）
        self.consume_thread = Thread(
            target=self.pose_consumer_loop,
            daemon=True
        )
    
    def angle_stream_loop(self):
        """生产者：读取舵机角度"""
        while not self.stop_event.is_set():
            arm_pos = []
            # 读取 7 个舵机
            for i in range(7):
                response = self.send_command(f'#{i:03d}PRAD!')
                angle = self.pwm_to_angle(response)
                arm_pos.append(np.radians(angle))
            
            # 发布到队列
            self.publish_arm_pos(arm_pos)
            time.sleep(1/50)  # 50 Hz
    
    def pose_consumer_loop(self):
        """消费者：控制仿真"""
        while not self.stop_event.is_set():
            # 从队列获取最新角度
            pose = self.get_latest_arm_pos()
            if pose is not None:
                # 转换为机器人动作
                action = self.convert_pose_to_action(pose)
                
                # 执行仿真
                self.env.step(action)
                self.env.render()
            
            time.sleep(1/50)  # 50 Hz
    
    def convert_pose_to_action(self, pose):
        """角度映射：舵机 → 机器人"""
        if self.robot_uids == "panda":
            # Panda 有 7 个手臂关节 + 2 个夹爪关节
            action = np.array(pose)
            action.insert(2, 0.0)  # 插入第 3 个关节
            action[-1] = self.angle_to_gripper(action[-1])
            return action
        
        elif self.robot_uids == "so100":
            # SO100 只有 5 个手臂关节
            action = np.array(pose[:5])
            action.append(self.angle_to_gripper(pose[6]))
            return action
        
        # ... 其他机器人映射
```

**工作流程**：

1. **生产者线程**：
   - 以 50 Hz 频率读取 7 个舵机角度
   - 将 PWM 值转换为角度（弧度）
   - 发布到线程安全队列

2. **消费者线程**：
   - 以 50 Hz 频率从队列获取最新角度
   - 根据机器人类型映射角度
   - 调用 `env.step()` 执行仿真
   - 调用 `env.render()` 渲染显示

3. **角度映射**：
   - 不同机器人关节数量不同
   - 需要处理关节顺序、方向
   - 夹爪角度需要特殊映射

## ⚙️ 物理引擎原理

### PhysX 物理引擎

**核心算法**：

#### 1. 刚体动力学

牛顿第二定律：
```
F = ma
τ = Iα

其中：
F - 力
m - 质量
a - 加速度
τ - 力矩
I - 转动惯量
α - 角加速度
```

**实现**：
```python
# 每个时间步
for body in bodies:
    # 计算合力
    total_force = gravity + contact_forces + applied_forces
    
    # 更新速度
    body.velocity += (total_force / body.mass) * dt
    
    # 更新位置
    body.position += body.velocity * dt
```

#### 2. 碰撞检测

**两阶段检测**：

1. **宽相位（Broad Phase）**：
   - 使用 AABB（轴对齐包围盒）快速排除
   - 空间分割（八叉树、网格）

2. **窄相位（Narrow Phase）**：
   - GJK 算法（凸体碰撞）
   - SAT 算法（分离轴定理）
   - 精确计算接触点和法线

```python
def detect_collision(body_a, body_b):
    # 宽相位：AABB 检测
    if not aabb_overlap(body_a.aabb, body_b.aabb):
        return None
    
    # 窄相位：精确检测
    contacts = gjk_algorithm(body_a.shape, body_b.shape)
    return contacts
```

#### 3. 约束求解

**关节约束**：

旋转关节（Revolute Joint）：
```
约束方程：C = θ - θ_target = 0

雅可比矩阵：J = ∂C/∂q

约束力：λ = -(JM⁻¹Jᵀ)⁻¹(J·v + bias)
```

**PD 控制器**：
```python
def pd_controller(target_pos, current_pos, current_vel, kp, kd):
    """PD 控制器计算力矩"""
    error = target_pos - current_pos
    torque = kp * error - kd * current_vel
    return torque
```

#### 4. 接触力计算

**库仑摩擦模型**：
```
法向力：Fn = k * penetration_depth
切向力：Ft ≤ μ * Fn

其中：
k - 接触刚度
μ - 摩擦系数
```

for each pixel in image:

### 着色器（Shader）

**顶点着色器**：
```glsl
#version 450

layout(location = 0) in vec3 position;
layout(location = 1) in vec3 normal;

uniform mat4 model;
uniform mat4 view;
uniform mat4 projection;

out vec3 frag_normal;
out vec3 frag_pos;

void main() {
    frag_pos = vec3(model * vec4(position, 1.0));
    frag_normal = mat3(transpose(inverse(model))) * normal;
    gl_Position = projection * view * vec4(frag_pos, 1.0);
}
```

**片段着色器**：
```glsl
#version 450

in vec3 frag_normal;
in vec3 frag_pos;

uniform vec3 light_pos;
uniform vec3 view_pos;
uniform vec3 object_color;

out vec4 frag_color;

void main() {
    // 环境光
    vec3 ambient = 0.1 * object_color;
    
    // 漫反射
    vec3 norm = normalize(frag_normal);
    vec3 light_dir = normalize(light_pos - frag_pos);
    float diff = max(dot(norm, light_dir), 0.0);
    vec3 diffuse = diff * object_color;
    
    // 镜面反射
    vec3 view_dir = normalize(view_pos - frag_pos);
    vec3 reflect_dir = reflect(-light_dir, norm);
    float spec = pow(max(dot(view_dir, reflect_dir), 0.0), 32);
    vec3 specular = 0.5 * spec * vec3(1.0);
    
    vec3 result = ambient + diffuse + specular;
    frag_color = vec4(result, 1.0);
}
```

## 🎮 控制系统

### PD 控制器

**数学模型**：
```
τ = Kp * (θ_target - θ_current) - Kd * θ̇_current

其中：
τ - 输出力矩
Kp - 比例增益（刚度）
Kd - 微分增益（阻尼）
θ_target - 目标角度
θ_current - 当前角度
θ̇_current - 当前角速度
```

**代码实现**：
```python
class PDJointPosController:
    def __init__(self, joints, stiffness, damping, force_limit):
        self.joints = joints
        self.kp = stiffness
        self.kd = damping
        self.force_limit = force_limit
    
    def compute_torque(self, target_qpos):
        # 获取当前状态
        current_qpos = self.get_qpos()
        current_qvel = self.get_qvel()
        
        # PD 控制
        error = target_qpos - current_qpos
        torque = self.kp * error - self.kd * current_qvel
        
        # 力矩限制
        torque = np.clip(torque, -self.force_limit, self.force_limit)
        
        return torque
    
    def step(self, target_qpos):
        torque = self.compute_torque(target_qpos)
        self.set_qf(torque)
```

### 末端执行器控制

**逆运动学（IK）**：

给定末端位置，求解关节角度：
```python
def inverse_kinematics(target_pos, target_rot):
    """数值优化方法求解 IK"""
    
    def objective(q):
        # 前向运动学
        fk_pos, fk_rot = forward_kinematics(q)
        
        # 位置误差
        pos_error = np.linalg.norm(target_pos - fk_pos)
        
        # 姿态误差
        rot_error = rotation_distance(target_rot, fk_rot)
        
        return pos_error + rot_error
    
    # 优化求解
    result = scipy.optimize.minimize(
        objective,
        x0=current_qpos,
        method='SLSQP',
        bounds=joint_limits
    )
    
    return result.x
```

## 🔧 性能优化

### GPU 并行仿真

**原理**：
```python
# CPU 仿真：串行
for env in envs:
    env.step(action)

# GPU 仿真：并行
# 所有环境同时在 GPU 上计算
gpu_envs.step(batched_actions)  # 一次性处理所有环境
```

**优势**：
- 1000+ 环境并行
- 显著加速训练
- 适合简单场景

**限制**：
- 不支持某些复杂物理特性
- 内存占用大

### 渲染优化

**技术**：
1. **视锥剔除**：只渲染可见物体
2. **LOD（细节层次）**：远处物体用低精度模型
3. **实例化渲染**：批量渲染相同物体
4. **延迟渲染**：分离几何和光照计算
