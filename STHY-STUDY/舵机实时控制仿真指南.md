# 🎮 舵机实时控制仿真指南

## 📋 概述

项目提供了 **`teleop_sim.py`** 程序，可以实时读取物理舵机（U-Arm）的角度，并同步控制仿真环境中的机器人。

### 系统架构

```
┌─────────────────┐      串口      ┌──────────────────┐
│  物理舵机硬件    │  ←─────────→  │  teleop_sim.py   │
│  (U-Arm)        │   115200 bps  │  (Python程序)     │
└─────────────────┘                └──────────────────┘
                                            ↓
                                   ┌──────────────────┐
                                   │  ManiSkill仿真    │
                                   │  (虚拟机器人)     │
                                   └──────────────────┘
```

---

## 🔧 硬件要求

### 必需硬件

1. **U-Arm 主操作臂**
   - 7个舵机（6个手臂关节 + 1个夹爪）
   - 舵机型号：中林/飞特舵机
   - 串口通信：115200 波特率

2. **USB转串口线**
   - 连接 U-Arm 到电脑
   - Linux 下通常识别为 `/dev/ttyUSB0`

3. **电源**
   - 为舵机供电（通常 12V）

### 可选硬件

- 如果没有硬件，可以修改代码模拟输入（见后文）

## 🚀 快速开始

### 步骤 1: 连接硬件

```bash
# 1. 连接 U-Arm 到电脑 USB 口

# 2. 检查串口设备
ls -l /dev/ttyUSB*

# 输出示例：
# /dev/ttyUSB0

# 3. 设置串口权限
sudo chmod 666 /dev/ttyUSB0

# 或者永久添加用户到 dialout 组
sudo usermod -a -G dialout $USER
# 需要重新登录生效
```

### 步骤 2: 运行仿真

```bash
cd src/simulation

# 基础用法（默认 SO-100 机器人）
python3 teleop_sim.py

# 指定机器人类型
python3 teleop_sim.py --robot panda

# 指定串口
python3 teleop_sim.py --robot xarm6_robotiq --serial-port /dev/ttyUSB1

# 完整参数
python3 teleop_sim.py \
    --robot panda \
    --scene ReplicaCAD_SceneManipulation-v1 \
    --serial-port /dev/ttyUSB0 \
    --rate 50.0
```

### 步骤 3: 操作

1. 程序启动后会自动校准舵机零位
2. 移动 U-Arm，仿真中的机器人会同步运动
3. 按 `Ctrl+C` 退出

## 🔍 工作原理详解

### 系统流程

```
┌──────────────────────────────────────────────────────────┐
│  1. 初始化阶段                                             │
│  • 打开串口 (115200 bps)                                  │
│  • 创建仿真环境                                            │
│  • 校准舵机零位                                            │
└──────────────────────────────────────────────────────────┘
                        ↓
┌──────────────────────────────────────────────────────────┐
│  2. 双线程运行                                             │
│                                                           │
│  ┌─────────────────────┐    ┌─────────────────────┐     │
│  │  生产者线程 (50Hz)   │    │  消费者线程 (50Hz)   │     │
│  │                     │    │                     │     │
│  │  读取舵机角度        │    │  从队列获取角度      │     │
│  │      ↓              │    │      ↓              │     │
│  │  PWM → 角度转换     │    │  角度映射转换        │     │
│  │      ↓              │    │      ↓              │     │
│  │  发布到队列          │    │  执行仿真步进        │     │
│  └─────────────────────┘    └─────────────────────┘     │
│           ↓                           ↓                  │
│      [线程安全队列]  ←──────────────→                     │
└──────────────────────────────────────────────────────────┘
                        ↓
┌──────────────────────────────────────────────────────────┐
│  3. 实时同步                                               │
│  • 物理舵机移动 → 仿真机器人同步移动                        │
│  • 延迟 < 20ms (50Hz 控制频率)                            │
└──────────────────────────────────────────────────────────┘
```

### 核心代码解析

#### 1. 生产者线程：读取舵机

```python
def angle_stream_loop(self):
    """以 50Hz 频率读取舵机角度"""
    while not self.stop_event.is_set():
        arm_pos = []
        
        # 读取 7 个舵机
        for i in range(7):
            # 发送命令读取角度
            response = self.send_command(f'#{i:03d}PRAD!')
            
            # PWM 值转换为角度
            angle = self.pwm_to_angle(response)
            
            # 减去零位，转换为弧度
            angle_rad = np.radians(angle - self.zero_angles[i])
            arm_pos.append(angle_rad)
        
        # 发布到队列
        self.publish_arm_pos(arm_pos)
        
        # 控制频率 50Hz
        time.sleep(1/50)
```

#### 2. 消费者线程：控制仿真

```python
def pose_consumer_loop(self):
    """以 50Hz 频率控制仿真"""
    while not self.stop_event.is_set():
        # 从队列获取最新角度
        pose = self.get_latest_arm_pos()
        
        if pose is not None:
            # 角度映射（舵机 → 机器人）
            action = self.convert_pose_to_action(pose)
            
            # 执行仿真步进
            self.env.step(action)
            self.env.render()
        
        # 控制频率 50Hz
        time.sleep(1/50)
```

#### 3. 角度映射示例

```python
def convert_pose_to_action(self, pose):
    """将 U-Arm 舵机角度映射到不同机器人"""
    
    if self.robot_uids == "panda":
        # Panda 有 7 个手臂关节
        action = np.array(pose)
        action.insert(2, 0.0)  # 插入第 3 关节（Panda 特有）
        
        # 调整关节方向
        action[3] = -action[3]
        action[4], action[5] = action[5], action[4]  # 交换 4、5
        
        # 夹爪映射
        action[-1] = self.angle_to_gripper(action[-1], -1.0, 1.0)
        
        return action
    
    elif self.robot_uids == "so100":
        # SO-100 只有 5 个手臂关节
        action = np.array(pose[:5])  # 只取前 5 个
        
        # 调整方向
        action[0] = -action[0]
        action[3] = -action[3]
        
        # 夹爪
        action.append(self.angle_to_gripper(pose[6], -1.1, 1.1))
        
        return action
    
    # ... 其他机器人映射
```

### 方案 1: 键盘控制

创建 `teleop_sim_keyboard.py`：

```python
import gymnasium as gym
import mani_skill.envs
import numpy as np
import time
from pynput import keyboard

class KeyboardTeleop:
    def __init__(self, robot_uids="panda"):
        self.env = gym.make(
            "Empty-v1",
            robot_uids=robot_uids,
            render_mode="human",
            control_mode="pd_joint_pos",
        )
        
        self.env.reset()
        
        # 当前关节角度
        self.current_qpos = np.zeros(8)
        
        # 控制增量
        self.delta = 0.05
        
        # 当前控制的关节
        self.active_joint = 0
        
        print("键盘控制:")
        print("  1-7: 选择关节")
        print("  +/-: 增加/减少角度")
        print("  r: 重置")
        print("  q: 退出")
    
    def on_press(self, key):
        try:
            if hasattr(key, 'char'):
                if key.char in '1234567':
                    self.active_joint = int(key.char) - 1
                    print(f"选择关节 {self.active_joint}")
                
                elif key.char == '+' or key.char == '=':
                    self.current_qpos[self.active_joint] += self.delta
                    print(f"关节 {self.active_joint}: {self.current_qpos[self.active_joint]:.2f}")
                
                elif key.char == '-':
                    self.current_qpos[self.active_joint] -= self.delta
                    print(f"关节 {self.active_joint}: {self.current_qpos[self.active_joint]:.2f}")
                
                elif key.char == 'r':
                    self.current_qpos = np.zeros(8)
                    print("重置")
                
                elif key.char == 'q':
                    return False
        except:
            pass
    
    def run(self):
        listener = keyboard.Listener(on_press=self.on_press)
        listener.start()
        
        try:
            while True:
                self.env.step(self.current_qpos)
                self.env.render()
                time.sleep(0.02)
        except KeyboardInterrupt:
            pass
        finally:
            self.env.close()

if __name__ == "__main__":
    teleop = KeyboardTeleop("panda")
    teleop.run()
```

### 方案 2: 鼠标/游戏手柄控制

使用 `pygame` 库读取游戏手柄输入：

```python
import pygame
import gymnasium as gym
import mani_skill.envs
import numpy as np

pygame.init()
pygame.joystick.init()

if pygame.joystick.get_count() == 0:
    print("未检测到游戏手柄")
    exit()

joystick = pygame.joystick.Joystick(0)
joystick.init()

env = gym.make("Empty-v1", robot_uids="panda", render_mode="human")
env.reset()

print(f"游戏手柄: {joystick.get_name()}")
print(f"轴数量: {joystick.get_numaxes()}")

qpos = np.zeros(8)

try:
    while True:
        pygame.event.pump()
        
        # 读取摇杆
        for i in range(min(6, joystick.get_numaxes())):
            axis_value = joystick.get_axis(i)
            qpos[i] = axis_value * 1.5  # 缩放到合适范围
        
        # 按钮控制夹爪
        if joystick.get_button(0):  # A 键
            qpos[7] = 0.04
        else:
            qpos[7] = 0.0
        
        env.step(qpos)
        env.render()
        
except KeyboardInterrupt:
    pass
finally:
    env.close()
    pygame.quit()
```

## 🐛 常见问题

### 1. 串口权限错误

```bash
# 错误: Permission denied: '/dev/ttyUSB0'

# 解决方案 1: 临时授权
sudo chmod 666 /dev/ttyUSB0

# 解决方案 2: 永久授权
sudo usermod -a -G dialout $USER
# 重新登录
```

### 2. 找不到串口设备

```bash
# 检查设备
ls -l /dev/ttyUSB*
ls -l /dev/ttyACM*

# 查看内核日志
dmesg | grep tty

# 安装驱动（如果需要）
sudo apt install linux-modules-extra-$(uname -r)
```

### 3. 舵机响应错误

```python
# 问题: 读取舵机返回空或错误数据

# 检查波特率
self.ser = serial.Serial('/dev/ttyUSB0', 115200)  # 确认是 115200

# 增加超时
self.ser = serial.Serial('/dev/ttyUSB0', 115200, timeout=0.1)

# 检查命令格式
response = self.send_command('#000PRAD!')  # 确认命令正确
```

### 4. 仿真延迟

```python
# 问题: 仿真跟不上舵机

# 降低控制频率
self.rate = 30.0  # 从 50Hz 降到 30Hz

# 简化场景
env = gym.make("Empty-v1", ...)  # 使用空场景

# 关闭光线追踪
sensor_configs=dict(shader_pack="default")  # 不用 rt-fast
```

### 5. 角度映射不对

```python
# 问题: 仿真机器人运动方向错误

# 调整关节方向
action[0] = -action[0]  # 反转关节 0

# 交换关节
action[3], action[4] = action[4], action[3]  # 交换 3 和 4

# 调试输出
print(f"舵机角度: {np.degrees(pose)}")
print(f"机器人动作: {np.degrees(action)}")
```

### 2. 减少抖动

```python
# 添加低通滤波
class LowPassFilter:
    def __init__(self, alpha=0.3):
        self.alpha = alpha
        self.prev = None
    
    def filter(self, value):
        if self.prev is None:
            self.prev = value
        self.prev = self.alpha * value + (1 - self.alpha) * self.prev
        return self.prev

# 使用
filter = LowPassFilter(alpha=0.3)
filtered_angle = filter.filter(raw_angle)
```
